# Frontend Context: Animaker

## Language
- **React**: UI library.
- **Lucide**: Icon library.
- **Tailwind**: Styling (though PRD prefers Vanilla CSS for new styles, existing uses Tailwind).
- **Edit Page**: The main workspace with Dual Viewers and Timeline.
- **Deliver Page**: The export interface.

## Glossary
- **Timeline**: The multi-track editing workspace.
- **Track**: A horizontal layer in the timeline (V1, V2, A1, etc.).
- **Clip**: A segment of media or animation on a track.
- **Inspector**: The panel for adjusting clip properties.
- **Viewer**: The video monitor (Source or Timeline).
