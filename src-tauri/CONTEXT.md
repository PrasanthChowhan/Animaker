# Backend Context: Animaker

## Language
- **Tauri**: Desktop framework.
- **Rust**: Systems language for core logic.
- **Hyperframes**: HTML-to-video rendering engine.
- **FFmpeg**: Media processing and composition.

## Glossary
- **Project Store**: The abstraction for saving/loading projects on disk.
- **Render Engine**: The module that orchestrates Hyperframes and FFmpeg.
- **Animation Clip**: A clip whose content is HTML/CSS/JS.
- **Media Clip**: A clip whose content is a video or audio file.
- **Composition**: The process of overlaying animation tracks on media tracks.
